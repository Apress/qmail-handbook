Page 69

In Listing 2-2 there is an error in the script.

 

The lines:

stop)

                 echo "Stopping qmail"

                 echo "  qmail-smtpd"

                 svc -d /service/qmail-smtpd 2>&1 | tee -a $LOG

                echo "  qmail-send"

                 svc -d /service/qmail-send 2>&1 | tee -a $LOG

                 if [ -f /var/lock/subsys/qmail ]; then

                         rm /var/lock/subsys/qmail

                 fi            echo "done"

 

Should read: 

stop)

                 echo "Stopping qmail"

                 echo "  qmail-smtpd"

                 svc -d /service/qmail-smtpd 2>&1 | tee -a $LOG

                 echo "  qmail-send"

                 svc -d /service/qmail-send 2>&1 | tee -a $LOG

                 if [ -f /var/lock/subsys/qmail ]; then

                         rm /var/lock/subsys/qmail

                 fi

                 echo "done"

 

Without making this change, the script will produce errors.
